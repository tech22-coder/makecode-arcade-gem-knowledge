# MakeCode Arcade Gem Knowledge

이 저장소는 MakeCode Arcade 블록 코딩을 배우는 학생들을 돕기 위한 Gemini Gem 지식 문서 모음이다.

## 목적

학생이 게임 제작 중 막혔을 때, 어시스턴트가 JavaScript 코드가 아니라 블록 코딩 기준으로 안내하도록 만든다.

## 포함 문서

- 00_gem_instructions.md: Gem의 기본 행동 지침
- 01_core_blocks.md: MakeCode Arcade 핵심 블록 정리
- 02_game_recipes.md: 게임 기능별 블록 배치 예시
- 03_extensions.md: 확장 프로그램 추천 목록
- 04_common_errors.md: 학생들이 자주 틀리는 점

## 답변 원칙

- 블록 카테고리 기준으로 설명한다.
- on start, on game update, 버튼 이벤트, overlap 이벤트처럼 배치 위치를 명확히 말한다.
- 학생에게 먼저 최소 기능을 만들게 하고, 이후 확장하게 한다.
- 복잡한 기능은 확장 프로그램을 추천한다.

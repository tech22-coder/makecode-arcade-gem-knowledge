# MakeCode Arcade 확장 프로그램 안내

확장 프로그램은 기본 블록만으로 만들기 어려운 기능을 쉽게 만들게 해준다.

기본 블록으로 가능한 기능은 먼저 기본 블록으로 안내한다.  
확장 프로그램은 기능이 복잡해질 때 추천한다.

## 확장 프로그램 추가 방법

학생에게는 다음 순서로 안내한다.

1. MakeCode Arcade 오른쪽 위 톱니바퀴를 누른다.
2. Extensions를 선택한다.
3. 검색창에 확장 이름이나 GitHub URL을 입력한다.
4. 확장을 추가한다.
5. 왼쪽 블록 목록에 새 카테고리가 생겼는지 확인한다.

---

## Status Bar

추천 상황:

- 체력바
- 보스 HP
- 마나바
- 에너지바

GitHub URL:

https://github.com/jwunderl/pxt-status-bar

학생 안내:

기본 변수로 체력을 만들 수 있지만, 화면에 막대로 보여주려면 Status Bar 확장이 적합하다.

---

## Character Animations

추천 상황:

- 걷기 애니메이션
- 점프 애니메이션
- 방향별 캐릭터 이미지 전환

GitHub URL:

https://github.com/microsoft/arcade-character-animations

학생 안내:

캐릭터가 움직일 때 자동으로 걷는 그림이 바뀌게 만들고 싶으면 Character Animations 확장을 사용한다.

## Mini Menu

- 시작 메뉴
- 선택지
- 일시정지 메뉴
- 상점 메뉴

GitHub URL:

https://github.com/riknoll/arcade-mini-menu

학생 안내:

게임 시작 화면이나 선택지 메뉴를 만들고 싶으면 Mini Menu 확장이 적합하다.
